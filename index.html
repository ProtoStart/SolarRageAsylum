Solar Rage Asylum

(Very Early Alpha version)

This game is in very early stages of development.
There isn't a playable game here yet, but a beta of the first basic level will be here soon (June 2021)
Want to join the team? Find our code on github or join the ProtoStart community subreddit

A 2D top down indy game about escaping an Asylum.

The game will be built and released incrementally over at least the remainder of 2021.
The first level will just have basic movement and very very simple to build stuff and is due out in June 2021. Once that is available we'll work on level 2. In level 2 you'll have to avoid guards, but they'll be a bit stupid and lazy - it makes it easy enough for level 2 of a game, and also means we can release level 2 without having made effective AI.

We'll add levels with new game mechanics and smarter, better equipped guards over time. By the end of 2021 there should be a pretty good game out of this - an indy game for sure, with creative game mechanics, tricky puzzles, some tense action and hopefuly an interesting story.

Oh and we'll have cheat codes and level codes, because I like old school stuff like that (and besides level codes are a quick and easy thing to get working, and don't require storing data)

Here's a very rough road map for the early months of development. Some things will overlap with previous activities

  1. Concept and requirements (DONE) - this was where I thought through the basics of what this game would be and how I'd do it
  2. Basic project setup (DONE) - staging area connected up to a github repo for the project, simple landing page built, concise description of what the game will be
  3. Main character design - (Started)
  4. Basic movement (Started)
  5. Simple first level with no guards
  6. 2nd level with stupid, short sighted guards
  7. Some more levels with increasingly better AI and more game mechanics
